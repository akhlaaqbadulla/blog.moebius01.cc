This is where I'll be writing about the things I work on — infrastructure, security,
cloud, AI deployment, and the open-source ecosystem that quietly runs everything underneath.

## What you'll find here

A few categories I expect to populate over time:

- **Infrastructure** — virtualization, IaC, hybrid architectures, design tradeoffs
- **Security** — hardening, posture management, audit-driven controls, real-world tooling
- **Cloud** — Azure, OVH, hybrid patterns, migration stories
- **AI / ML** — running local models in production (Ollama, vLLM), MCP, agentic workflows
- **Open Source** — drop-in replacements for proprietary corporate tooling
- **DevOps** — automation, pipelines, Terraform and Ansible patterns

## How I write

Short, opinionated, grounded in things I've actually shipped. Less "what is Kubernetes",
more "here's the specific pain we hit running it in a multi-site environment, and what we
did about it."
